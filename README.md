# Tomato with RemoteMonster 

> Concentration Management Online Study With WebRTC([RemoteMonster](https://remotemonster.com/))

## Quick Start

### Git Clone
```
# git clone https://github.com/Koozzi/Tomato.git
```

### Installations

```
Tomato # npm install

create-react-app
express
mongoose
concurrently
nodemon

Tomato/client # npm install

bootstrap
reactstrap
@remotemonster/sdk
react-router-dom
react-helmet
react-script-tag
@material-ui/icons
@material-ui/core
```

### Start
```
# npm run dev
```
